# 八戸市人事データ抽出プログラム 手順書

本ドキュメントは、Excel形式の「人事異動内示データ」から必要な情報をルールに従ってテキスト（`.txt`）形式で抽出するプログラムの使用手順をまとめたものです。

---

## 1. 準備事項

このプログラムはPythonで作られており、`uv` コマンドで実行します。
初回実行時や、パソコンを買い替えた際などは、以下のフォルダが存在することを確認してください。

* **プログラムフォルダの場所**: `C:\Users\tked1\py\hachinohe_personnel_data`
* このフォルダの中に `extract_personnel.py` （抽出用プログラム）が入っています。

---

## 2. 実行手順

1. **コマンドプロンプト（またはターミナル）を開く**
   Windowsの「スタートメニュー」から `cmd` または `Terminal` を検索し、コマンドプロンプトを起動します。

2. **プログラムのフォルダへ移動する**
   以下のコマンドをコピーして貼り付け、Enterキーを押します。

   ```bash
   cd c:\Users\tked1\py\hachinohe_personnel_data
   ```

3. **プログラムを実行する**
   以下のコマンドをコピーして貼り付け、Enterキーを押します。

   ```bash
   uv run extract_personnel.py
   ```

4. **エクセルファイルを選択する**
   * プログラムを実行すると、数秒後に「**抽出する人事データのエクセルファイルを選択してください**」という画面（ファイル選択ダイアログ）がピコンと開きます。（画面の後ろに隠れてしまうことがあるので、タスクバーのアイコンなどを確認してください）
   * 抽出したい人事データのエクセルファイル（`.xlsx`）を探して選択し、「開く」ボタンをクリックします。
   * ※抽出処理をキャンセルしたい場合は、ここで「キャンセル」や×ボタンを押せば処理が止まります。

5. **抽出結果を確認する**
   * コマンドの画面に `抽出が完了しました: C:\Users\tked1\py\hachinohe_personnel_data\〇〇_抽出結果.txt` と表示されれば完了です。
   * プログラムと同じフォルダ（`c:\Users\tked1\py\hachinohe_personnel_data`）の中に、**元ファイル名に「_抽出結果」がついたテキストファイル** が新しく出来上がっていますので、ダブルクリックして中身を確認してください。

## 3. 本プログラムの抽出ルールについて

このプログラムは、以下のルールに従って自動的に文字を補完・置換・抽出しています。

* **局・級ヘッダ表示**: 各ページの先頭にある「[市長事務部局]」や「< 部長級 >」などの表示は、データが変わったタイミングのみ出力します。
* **部名（B列）の自動補完**: B列が空欄になっている行でも、直前に登場した部名を自動的に補って出力します。ただし、級名が「部長級」「次長級」の際、および「出向」の際には文字の重複を避けるためB列の補完は行いません。
* **不用記号の削除・置換**:
  * データ内の読点「、」やカッコ「()」「（）」はすべて中点「・」に変換されます。
  * 「㈱」などの文字は自動的に削除されます。
* **氏名（D列）内の全角/半角スペース**: そのまま維持されて出力されます。
* **旧職名（E列）の処理**: 中身のカッコはすべて「・」に変換し、先頭と末尾にだけ全角の「（）」がつくように加工されます。ただし、A列が「再任用」や「採用」で旧職がない人については「　」（全角スペース）が挿入されます。
* **異動区分の処理分岐**:
  * **退職**: 「退職・３１日[新職先]（旧職）氏名」という形式になります。
  * **出向など（転任・昇任・転任/昇任・配置換・兼務以外）**: C列（新職名）のあとに、A列（異動区分）が「・」で繋がれます（例：`新職名・出向（旧職）氏名`）。
  * **降任（役職定年）**: 基本的に『出向など』と同じ扱いですが、「降任」が省かれ「・役職定年」という出力になります。
