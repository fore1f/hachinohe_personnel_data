# Task

## 目的
八戸市の人事異動内示データ（エクセルファイル）を読み込み、指定された条件・ルールに基づいて個人の異動情報をテキスト形式で抽出・整形するプログラムを作成する。

## 要件
*   **ファイル選択**: ファイル選択ダイアログを使用し、複数のエクセルファイルから処理対象をユーザーが選べるようにする。
*   **出力**: 選択されたファイル名をもとに「〇〇_抽出結果.txt」というファイル名で同一フォルダ内にテキストデータを出力する。
*   **データ整形ルール**:
    *   ページごとに局名・級名（「< 部長級 >」など）が前ページから変わった場合のみ出力する。
    *   「異動区分」「新職名」「氏名」「旧職名」等のヘッダ行・不要行はスキップする。
    *   B列（部名）は直前のデータを記憶して不足分を補う。ただし、「部長級」「次長級」の場合はB列の結合を省く。
    *   氏名（D列）内の空白は維持する。
    *   文字データの全角・半角スペースや読点「、」は置換（削除または「・」へ変換）し、さらに「（）」等も「・」に変換する。
    *   旧職（E列）の両端には必ず全角の「（）」を付与する。ただし、異動区分が「再任用」や「採用」で旧職が存在しない場合は全角スペース「　」を付与する。
*   **個別パターンの出力スタイル**:
    *   **退職**: `退職・[日付][新職への出向等]（[旧職]）[氏名]`
        *   日付は月が処理月と同じなら「〇〇日」、違えば「〇〇月〇〇日」と全角数字で出力する。
    *   **出向（転任等以外）**: `[B列（部名省略）][C列の新職]・[A列の出向等]（[E列の旧職]）[D列の氏名]`
        *   「降任（役職定年）」は「・役職定年」に置き換える。
    *   **基本抽出**: `[B列][C列]（[E列の旧職]）[D列の氏名]`
